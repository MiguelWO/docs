## Bienvenido

Esta API está diseñada para facilitar la gestión de agentes, inmuebles y clientes en una inmobiliaria. Se pueden realizar operaciones como crear, actualizar, eliminar y consultar datos relacionados con estos recursos.

Utilizamos el estándar [OpenAPI](https://mintlify.com/docs/api-playground/openapi/setup) para garantizar una documentación clara y funcional.

## Autenticación

Todas las rutas de la API requieren autenticación mediante un encabezado con una API Key. Incluye el encabezado `x-api-key` en cada solicitud para acceder a los recursos.

```json
"security": [
  {
    "x-api-key": []
  }
]
