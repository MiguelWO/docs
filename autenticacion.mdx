## Introducción a la Autenticación

La autenticación es fundamental para proteger el acceso a nuestra API. Garantizamos la seguridad de tus datos mediante un robusto sistema de autenticación basado en api tokens.

### Características Clave

- **Seguridad Robusta**: Implementamos el estándar API key
- **Gestión Simple**: Proceso de obtención de token claro y directo
- **Tokens Temporales**: Seguridad adicional mediante tokens con tiempo de vida limitado

### 2. Solicitud de Token de Acceso

Utiliza tus credenciales para obtener un token de autenticación:

```bash
curl -X POST "https://api.cerouno.ai/v1/auth/login" \
-H "Content-Type: application/json" \
-d '{
  "usuario": "tu_usuario",
  "contrasena": "tu_contraseña"
}'
```

#### Respuesta de Ejemplo

```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expiracion": "2024-12-31T23:59:59Z"
}
```

### 3. Uso del Token en Solicitudes

Incluye el token en el encabezado `x-api-key` de cada petición:

```bash
curl -X GET "https://api.cerouno.ai/v1/inmuebles" \
-H "x-api-key : TU_TOKEN_DE_ACCESO"
```

## Solución de Problemas

<Accordion title="Errores Comunes de Autenticación">
  - **401 Unauthorized**: Token inválido o expirado
  - **403 Forbidden**: Permisos insuficientes
  - **400 Bad Request**: Formato de solicitud incorrecto
</Accordion>
