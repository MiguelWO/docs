## Bienvenidos

Bienvenidos a la documentación de API de Cerouno. Hemos trabajado para hacer de esta API algo muy fácil de integrar y trabajar. En Cerouno nos soñamos cambiando y mejorando la manera como las personas encuentran su próximo hogar en Colombia. Nuestra tecnología nos pone un paso más cerca de lograrlo, y por ello nos esforzamos en hacer esta documentación intuitiva y funcional.

Por el momento, podrás integrarte con nosotros como software inmobiliario o como inmobiliaria para:
- Enviarnos la información de tu inventario.
- Actualizar y gestionar tus propiedades.
- Organizar los agentes encargados de las propiedades.

Queremos que nuestra API crezca y se convierta en una fuente de información crucial en el sector inmobiliario. Si tienes ideas para mejorarla, ¡las estaremos esperando!

Vamos a estar siempre atentos para que la experiencia trabajando con la API de Cerouno sea un proceso fácil y eficiente. Siempre estaremos a un contacto de distancia. Así que bienvenido, esperamos que nos acompañes en el futuro de ayudar a las personas a encontrar su próximo hogar.
