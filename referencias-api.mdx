## Endpoints Principales

<Accordion title="Lista de Endpoints Disponibles">
  ### Propiedades
  - `GET /api/listings`: Listar propiedades
  - `GET /api/listings/{id}`: Consultar propiedad por ID
  - `POST /api/listings`: Crear nueva propiedad
  - `PUT /api/listings/{id}`: Actualizar propiedad
  - `DELETE /api/listings/{id}`: Eliminar propiedad

  ### Agentes
  - `GET /api/agents`: Listar agentes
  - `GET /api/agents/{id}`: Consultar agente por ID
  - `GET /api/agents/broker/{id}`: Consultar agentes por broker
  - `POST /api/agents`: Crear nuevo agente
  - `PUT /api/agents/{id}`: Actualizar agente
  - `DELETE /api/agents/{id}`: Eliminar agente

  ### Inmobiliarias
  - `GET /api/clients`: Listar inmobiliarias
  - `GET /api/clients/{id}`: Consultar inmobiliaria por ID
  - `POST /api/clients`: Crear nueva inmobiliaria
  - `PUT /api/clients/{id}`: Actualizar inmobiliaria
  - `DELETE /api/clients/{id}`: Eliminar inmobiliaria
  
</Accordion>

## Autenticación y Seguridad

<Steps>
  <Step title="Obtener Credenciales">
    Contacta con nuestro equipo para recibir tus credenciales de acceso
  </Step>

  <Step title="Solicitar Token">
    Realiza una solicitud POST al endpoint de autenticación
  </Step>

  <Step title="Incluir Token">
    Añade el token recibido en el encabezado `x-api-key`
  </Step>
</Steps>

### Ejemplo de Solicitud Autenticada

```bash
curl -X GET "https://api.cerouno.ai/v1/propiedades" \
-H "x-api-key: TU_TOKEN_DE_ACCESO"
```

<Tip>
  💡 Consejos para una Integración Exitosa:
  - Maneja siempre los códigos de respuesta
  - Implementa reintentos con backoff exponencial
  - Valida los datos antes de enviarlos
  - Mantén tus credenciales seguras
</Tip>
