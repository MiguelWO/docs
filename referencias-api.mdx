## Endpoints Principales

<Accordion title="Lista de Endpoints Disponibles">
  ### Propiedades
  - `GET /v1/propiedades`: Listar propiedades
  - `POST /v1/propiedades`: Crear nueva propiedad
  - `PUT /v1/propiedades/{id}`: Actualizar propiedad
  - `DELETE /v1/propiedades/{id}`: Eliminar propiedad

  ### Agentes
  - `GET /v1/agentes`: Listar agentes
  - `POST /v1/agentes`: Crear nuevo agente
  - `PUT /v1/agentes/{id}`: Actualizar agente
  - `DELETE /v1/agentes/{id}`: Eliminar agente

  ### Inventario
  - `GET /v1/inventario`: Consultar inventario
  - `POST /v1/inventario`: Actualizar inventario
</Accordion>
