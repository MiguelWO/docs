## Endpoints Principales

<Accordion title="Lista de Endpoints Disponibles">
  ### Propiedades
  - `GET /v1/propiedades`: Listar propiedades
  - `POST /v1/propiedades`: Crear nueva propiedad
  - `PUT /v1/propiedades/{id}`: Actualizar propiedad
  - `DELETE /v1/propiedades/{id}`: Eliminar propiedad

  ### Agentes
  - `GET /v1/agentes`: Listar agentes
  - `POST /v1/agentes`: Crear nuevo agente
  - `PUT /v1/agentes/{id}`: Actualizar agente
  - `DELETE /v1/agentes/{id}`: Eliminar agente

  ### Inventario
  - `GET /v1/inventario`: Consultar inventario
  - `POST /v1/inventario`: Actualizar inventario
</Accordion>

## Autenticación y Seguridad

<Steps>
  <Step title="Obtener Credenciales">
    Contacta con nuestro equipo para recibir tus credenciales de acceso
  </Step>

  <Step title="Solicitar Token">
    Realiza una solicitud POST al endpoint de autenticación
  </Step>

  <Step title="Incluir Token">
    Añade el token recibido en el encabezado `Authorization`
  </Step>
</Steps>

### Ejemplo de Solicitud Autenticada

```bash
curl -X GET "https://api.cerouno.ai/v1/propiedades" \
-H "Authorization: Bearer TU_TOKEN_DE_ACCESO"
```

## Consideraciones Importantes

<Tip>
  💡 Consejos para una Integración Exitosa:
  - Maneja siempre los códigos de respuesta
  - Implementa reintentos con backoff exponencial
  - Valida los datos antes de enviarlos
  - Mantén tus credenciales seguras
</Tip>
